Savestates.

Загрузка из cas, возможно из rom (в нем нет с какого адреса грузить в память и
стартовать, придется догадываться).

Реализовать свои форматы pmi (memory image), pcs (cassette), ptp (tape), pwe
(wave edges), pss (savestate). Насчет pss стоит подумать о целесообразности,
ибо libretro уже содержит в себе возможности сохранения и т.н. "сериализации".

Примеры расширений:
.img.pk8k  .img.8k  .i8k
.cas.pk8k  .cas.8k  .c8k
.edg.pk8k  .edg.8k  .e8k
.sav.pk8k  .sav.8k  .s8k

При подгрузке образа, если это ROM и он начинается и запускается с 0, загружать
его вместо штатного bios. В таком образе должна быть возможность задавать карту
точек входа в bios для перехвата ввода-вывода на ленту.

Т.н. code chunks - куски кода для исполнения вне пространства RAM/ROM/Х - для
наиболее корректной поддержки BIOS-хуков
